mxBB-Portal Online Help

1 . General->Configuration

This panel is where most of the portal settings and preferences can be changed:

 

  1. General Settings:

Portal Name - here the Admin can change the portal name;


Enable portal - Handy switch, when reconstructing the site. Only admin is able to view pages and browse around normally. While disabled, the message below is displayed. Portal disabled message We are currently upgrading this site with latest mxBB software.


URL to mxBB Portal - Full Portal URL, ex. http://www.example.com/http://www.example.com/

 

URL to your phpBB installation - Full phpBB URL, ex. http://www.example.com/phpBB/http://www.example.com/phpBB/


Use mxBB Block Cache - Block data is cached to individual cache/block_*.xml/php files. Block cache files are created/updated when blocks are edited.


Use mod_rewrite -If you're running on an Apache server and have mod_rewrite activated, you may rewrite URLS; for example, you can rewrite pages like 'page=x' with more intuitive alternatives. Please read further documentation for the mx_mod_rewrite module.

 

  1. Style Settings:

 

AdminCP Style - The default AdminCP Style.

 

Portal Pages Style - The default style used in portal pages, this can be overwritten from pageCP for each page.


Override user style - Replaces users style with the default (for pages).


Overall Header File (default value) - This is the default template overall_header file, e.g. overall_header.tpl.


Overall Footer File (default value) - This is the default template overall_footer file, e.g. overall_footer.tpl.


Main Layout File (default value) - This is the default template main_layout file, e.g. mx_main_layout.tpl.


Overall Navigation Block (default value) - This is the page header navigation block, provided you've chosen a overall header file which supports page navigation.

Overall Navigation Block - Chose the overall navigation block.

phpBB Statistics in Header (default value) - Links to new, unread posts in header.
