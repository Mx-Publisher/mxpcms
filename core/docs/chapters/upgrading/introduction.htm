Introduction

As an administrator, it is always best for you to keep all software installations on your server as up-to-date as possible. New versions may have security and bug fixes that you need to have. Fortunately, upgrading is very easy. In order to upgrade from the mxBB 2.7 series to mxBB 2.8.1, you must be using the latest version of mxBB 2.7.

Download Packages

There are normally three different packages that you can use to upgrade your mxBB installation: Full, Changed Files, and Code Changes. However, if you are upgrading from the mxBB 2.7 series, the only option is the Full package, due to the enormity of changes between mxBB 2.7.x and mxBB 2.8.1.

All official upgrade packages can be found in this category at our main site.

In order to upgrade from the mxBB 2.7 series to mxBB 2.8.1, please download the Full package.

In order to upgrade from the mxBB 2.8.0 to mxBB 2.8.1, upload the Full package except config.php.

Before proceeding, make backups of everything: the files on your site as well as the databases available. Store these backups in a safe location.
