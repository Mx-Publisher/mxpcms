Description
The MX Module is an addon product for the mxBB-portal. This module / block is showing latest news and recent topics on any page.Customizable to show only news, only recent topics, or both. Will check the authorization of users and only show allowed topics.

Features
Easy to use and install/uninstall module.

I. Installation Instructions
To install this module, follow these instructions.

To summarize:
If installing from scratch, use import module in the AdminCP.
If upgrading, upgrade from AdminCP
If uninstalling, use the delete feature in the AdminCP

III. Additional Languages
First check to see if your language is already translated.

If exists, download and install in the modules/mx_glance/language folder. If not, duplicate (copy and paste) any included language file, rename to match your language, translate using any texteditor, save and upload.
