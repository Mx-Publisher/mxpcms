Description
The MX Module is an addon product for the MX-portal. This is the shotcast version of mx_radio. The player can play one stream brotcasted by a shotcast server that can be configured in AdminCP (host, port and password). The embed's are hidden in iframes with autostarts, so the pricipal is very simple. It also uses an ajax script to update the song title. Users most have instaled the latest media player (Windows Media Player 10+) or Real One Player, but Real player version doesnt not worked with Firefox and Netscape and should be fixed now.

I. Installation Instructions
To install this module, follow these instructions.

To summarize:
If installing from scratch, use import module in the AdminCP.
If upgrading, uninstall, then reinstall from AdminCP
If uninstalling, use the delete feature in the AdminCP

Note: If upgrading,uninstall module without delting db entries, or else you'll lose all module data!

III. Additional Languages
First check to see if your language is already translated.

If exists, download and install in the modules/mx_shotcast/language folder. If not, duplicate (copy and paste) any included language file, rename to match your language, translate using any texteditor, save and upload.
